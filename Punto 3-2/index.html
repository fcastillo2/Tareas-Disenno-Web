¿Qué es el Diseño Web Responsivo?

El Diseño Web Responsivo (o Responsive Web Design) es un enfoque de diseño y desarrollo web que consiste en crear un único sitio web capaz de adaptar su estructura, imágenes y contenidos de forma automática al tamaño de pantalla y resolución de cualquier dispositivo

Viewport

El viewport es el área visible de una página web en la pantalla del usuario. En un dispositivo móvil, si no se dan las instrucciones correctas, los navegadores intentan renderizar el sitio como si se estuviera viendo en un monitor de escritorio, obligando al usuario a hacer zoom constantemente o a desplazarse de forma horizontal

Media Queries

Las media queries son una función fundamental de CSS3 que hace posible este diseño adaptable. Funcionan como condiciones lógicas ("si se cumple X, aplica Y") que detectan el entorno del usuario.
